Convert each raw manifest into a Helm template. Thin wrappers preserving every field. Hardcode `namespace: pihole` everywhere (do NOT use `{{ .Release.Namespace }}`).

1. `templates/namespace.yaml`:
```yaml
apiVersion: v1
kind: Namespace
metadata:
  name: pihole
```

2. `templates/daemonset.yaml` — Critical template. Must preserve ALL of:
   - `hostNetwork: {{ .Values.hostNetwork }}`
   - `dnsPolicy: {{ .Values.dnsPolicy }}`
   - `affinity` block using `.Values.nodeAffinity.key` and `.Values.nodeAffinity.values`:
     ```yaml
     affinity:
       nodeAffinity:
         requiredDuringSchedulingIgnoredDuringExecution:
           nodeSelectorTerms:
             - matchExpressions:
                 - key: {{ .Values.nodeAffinity.key }}
                   operator: In
                   values:
                     {{- toYaml .Values.nodeAffinity.values | nindent 20 }}
     ```
   - Container image: `{{ .Values.image.repository }}:{{ .Values.image.tag }}`
   - `imagePullPolicy: {{ .Values.image.pullPolicy }}`
   - Env vars: iterate over `.Values.env` map to produce name/value pairs:
     ```yaml
     env:
       {{- range $key, $val := .Values.env }}
       - name: {{ $key }}
         value: {{ $val | quote }}
       {{- end }}
     ```
   - `securityContext.capabilities.add: [NET_ADMIN]` — hardcoded, not templatized
   - All 4 ports: 53/UDP (dns-udp), 53/TCP (dns-tcp), 67/UDP (dhcp), 8080/TCP (http)
     Port 8080 should be templatized as `{{ .Values.service.webPort }}` for the http port
   - Resources using `{{ toYaml .Values.resources | nindent 12 }}`
   - Labels `app: pihole` on metadata and selector

3. `templates/service-web.yaml`:
```yaml
apiVersion: v1
kind: Service
metadata:
  name: pihole-web
  namespace: pihole
spec:
  selector:
    app: pihole
  type: ClusterIP
  ports:
    - name: http
      port: {{ .Values.service.webPort }}
      targetPort: {{ .Values.service.webPort }}
      protocol: TCP
```

4. `templates/ingress.yaml` — Must use Traefik CRD:
```yaml
apiVersion: traefik.io/v1alpha1
kind: IngressRoute
metadata:
  name: pihole
  namespace: pihole
spec:
  entryPoints:
    - web
  routes:
    - match: Host(`{{ .Values.ingress.host }}`)
      kind: Rule
      services:
        - name: pihole-web
          port: {{ .Values.service.webPort }}
```

After creating all templates, run `helm template charts/pihole` to verify rendering. Fix any template errors. Pay special attention to the env var iteration — the order may differ from the original manifest but all vars must be present.
