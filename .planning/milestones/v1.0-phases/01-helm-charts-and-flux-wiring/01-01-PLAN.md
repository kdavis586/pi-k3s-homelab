Convert each raw manifest into a Helm template. These are thin wrappers — preserve every field verbatim, only templatize the values extracted into values.yaml. Hardcode `namespace: jellyfin` in every template (do NOT use `{{ .Release.Namespace }}`).

1. `templates/namespace.yaml`:
```yaml
apiVersion: v1
kind: Namespace
metadata:
  name: jellyfin
```

2. `templates/deployment.yaml` — This is the critical template. Must preserve ALL of:
   - `strategy.type: Recreate`
   - `nodeSelector` using `{{ toYaml .Values.nodeSelector | nindent 8 }}`
   - Container image: `{{ .Values.image.repository }}:{{ .Values.image.tag }}`
   - `imagePullPolicy: {{ .Values.image.pullPolicy }}`
   - Port 8096 named `http`
   - Env var `JELLYFIN_PublishedServerUrl` with value `{{ .Values.publishedServerUrl }}`
   - Resources using `{{ toYaml .Values.resources | nindent 12 }}`
   - Volume `config` as PVC with `claimName: jellyfin-config`
   - Volume `media` as hostPath with `path: {{ .Values.storage.mediaHostPath }}` and `type: DirectoryOrCreate`
   - Liveness probe: httpGet /health:8096, initialDelaySeconds 60, periodSeconds 30, failureThreshold 5
   - Readiness probe: httpGet /health:8096, initialDelaySeconds 30, periodSeconds 10
   - NO securityContext at pod level, NO fsGroup anywhere
   - Labels `app: jellyfin` on metadata and selector

3. `templates/service.yaml`:
```yaml
apiVersion: v1
kind: Service
metadata:
  name: jellyfin
  namespace: jellyfin
  labels:
    app: jellyfin
spec:
  selector:
    app: jellyfin
  ports:
    - name: http
      port: 8096
      targetPort: 8096
      protocol: TCP
  type: ClusterIP
```

4. `templates/ingress.yaml` — Must use Traefik CRD, NOT standard Ingress:
```yaml
apiVersion: traefik.io/v1alpha1
kind: IngressRoute
metadata:
  name: jellyfin
  namespace: jellyfin
spec:
  entryPoints:
    - web
  routes:
    - match: {{ $matches := list }}{{ range .Values.ingress.hosts }}{{ $matches = append $matches (printf "Host(`%s`)" .) }}{{ end }}{{ join " || " $matches }}
      kind: Rule
      services:
        - name: jellyfin
          port: 8096
```
NOTE: The match template must produce the exact pattern: `Host("jellyfin.local") || Host("192.168.1.100") || Host("192.168.1.101") || Host("192.168.1.102")`. Use backticks inside the Host() function as Traefik requires them.

5. `templates/pvc.yaml`:
```yaml
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: jellyfin-config
  namespace: jellyfin
spec:
  accessModes:
    - ReadWriteOnce
  storageClassName: {{ .Values.storage.storageClassName }}
  resources:
    requests:
      storage: {{ .Values.storage.configSize }}
```

After creating all templates, run `helm template charts/jellyfin` to verify rendering. Fix any template errors.
